## Overview

Sketch Advanced is the leading native design application for UI and UX design that established the component-based approach to interface design and continues to serve as the primary tool for product designers creating interfaces, icons, and design systems.

The component system with Symbols allows design elements to be defined once and reused across any number of artboards, with overrides that let individual instances vary from the master while all instances update automatically when the master changes.

Auto Layout makes designs respond to content changes automatically, resizing and repositioning elements based on defined constraints and spacing rules, making it practical to design components that behave correctly for variable-length content.

Prototyping tools connect artboards with interactive hotspots and transition animations, producing clickable prototypes that demonstrate application flow without requiring a separate prototyping application.

For product designers, UI designers, and design teams who work on digital interfaces and want a native, purpose-built design tool with the deepest component system and the largest ecosystem of plugins and resources, Sketch Advanced is the design environment that defined modern UI design practice.

## Performance

Sketch balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
